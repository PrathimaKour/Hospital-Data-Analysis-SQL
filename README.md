# Hospital Data Analysis SQL Project

## Overview
This project analyzes hospital operational data using MySQL.

The dataset contains:
- Hospital Names
- Locations
- Departments
- Doctors Count
- Patients Count
- Admission and Discharge Dates
- Medical Expenses

## SQL Concepts Used
- Aggregate Functions
- GROUP BY
- ORDER BY
- LIMIT
- DATEDIFF()
- STR_TO_DATE()
- Subqueries
- Window Functions

## Business Problems Solved
1. Total Number of Patients
2. Average Doctors per Hospital
3. Top Departments by Patients
4. Hospital with Highest Expenses
5. Daily Average Expenses
6. Longest Hospital Stay
7. Patients Treated Per City
8. Average Stay Per Department
9. Department with Lowest Patients
10. Monthly Expense Report

## Tools Used
- MySQL
- MySQL Workbench
- CSV Dataset
